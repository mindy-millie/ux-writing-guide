# 보이스

> **담당자**: @Evelyn (에블린/무제한독서팀)
>
> 보이스(Voice)는 밀리의서재가 항상 유지하는 브랜드의 일관된 성격입니다. 상황이 달라져도 목소리는 바뀌지 않습니다. 사용자에게 어떤 말을 건네든 아래 네 가지 성격을 지킵니다.

## 1. 명확하게 말합니다 (Clear)

사용자가 생각하지 않아도 이해할 수 있도록 쉽고 직관적으로 말합니다. 특히 중요한 정보는 정확한 어휘와 일관된 표현으로 오해 없이 전달합니다.

**이렇게 합니다**

- 어려운 표현보다 쉬운 단어를 사용합니다.
- 한 번에 이해할 수 있는 문장 구조를 만듭니다.
- 불필요한 설명을 걷어내고 핵심 정보만 전달합니다.
- 현재 상태와 다음 행동을 짧고 분명하게 안내합니다.

| ✅ Do | ❌ Don't |
| --- | --- |
| "지금 읽고 있는 책이에요" | "현재 열람 중인 콘텐츠입니다" |
| "이어 읽기" | "독서 진행을 이어가세요" |
| "이용할 수 없는 페이지예요" | "잘못된 접근입니다" |

> 💡 문제가 생긴 순간 사용자는 이미 당황해 있어요. 밀리는 거기서 더 놀라게 하지 않습니다. 차분하게 상황을 설명하고, 다음에 무엇을 하면 되는지 바로 알려줘요.

## 2. 배려하며 말합니다 (Considerate)

사용자의 맥락과 감정을 고려해 말합니다. 무엇이 안 되는지보다 지금 할 수 있는 행동을 먼저 안내하여 불편함을 줄여줍니다.

**이렇게 합니다**

- 사용자의 상황에 공감하고, 해결책을 자연스럽게 제시합니다.
- 무엇이 안 되는지보다 지금 할 수 있는 것을 먼저 알려줍니다.
- 강요하거나 명령하지 않고 부드럽게 제안합니다.
- 사용자에게 책임을 돌리는 표현을 피합니다.

| ✅ Do | ❌ Don't |
| --- | --- |
| "입력한 정보를 다시 확인해 주세요" | "입력 값이 올바르지 않습니다" |
| "잠시 후 다시 시도해 주세요" | "오류가 발생했습니다" |
| "이용할 수 없는 서비스예요" | "사용 권한이 없습니다" |

> 💡 독서는 귀찮음 또는 부담스러움 같은 감정으로 연결될 수 있는 행동이에요. 밀리는 독서를 강요하거나 재촉하지 않아요. 대신 사용자의 감정에 먼저 다가가고 자연스럽게 다음 행동을 제안합니다.

## 3. 친근하게 말합니다 (Friendly)

서비스가 아닌 사람이 말하는 것처럼 자연스럽고 편안하게 표현합니다. 옆에서 같이 책을 고르는 친구처럼 다정하지만 가볍지 않게 말합니다.

**이렇게 합니다**

- 일상에서 익숙한 표현으로 거리감을 줄입니다.
- 기계적인 설명 대신 대화하듯 안내합니다.
- 부드러운 구어체와 해요체를 기본으로 사용합니다.
- 친근함을 유지하되, 줄임말·유행어·불필요한 기호는 쓰지 않습니다.
- 가르치거나 설득하지 않고, 자연스럽게 함께합니다.

| ✅ Do | ❌ Don't |
| --- | --- |
| "오늘 좋은 책이 들어왔어요" "읽어볼까요?" | "공개 예정 도서 지금 확인해보세요" |
| "다시 시도해 주세요" | "진행하시겠습니까?" "요청을 재시도 바랍니다" |

> 💡 책은 어렵다는 이미지가 있어요. 밀리는 그 거리를 친근함으로 좁힙니다. 사용자의 취향을 존중하고, 독서를 부담 없이 시작할 수 있도록 응원합니다.

## 4. 믿음직하게 말합니다 (Trustworthy)

구독 서비스에서 신뢰는 가장 중요한 기반입니다. 서비스 정책과 시스템 상황을 숨기거나 모호하게 표현하지 않고, 정확한 정보와 이유를 전달합니다.

**이렇게 합니다**

- 과장된 표현을 지양하고 정확한 사실을 전달합니다.
- 서비스 정책과 상태를 모호하게 숨기지 않습니다.
- 사용자가 서비스 동작을 이해하고 신뢰할 수 있도록 명확한 기준을 제공합니다.
- 책을 서비스하는 브랜드로서 가볍지 않은 이미지를 유지합니다.

| ✅ Do | ❌ Don't |
| --- | --- |
| "구독이 시작됐어요" | "구독 신청이 완료됐습니다" |
| "다음 결제일은 5월 1일이에요" | "자동 갱신 일정이 도래했습니다" |
| "이 콘텐츠는 구독 후 이용할 수 있어요" | "접근 권한이 없습니다" |

> 💡 밀리는 정보를 숨기지 않아요. 구독, 결제, 혜택 등 사용자의 권리와 선택의 결과를 명확하게 알립니다. 신뢰는 투명한 언어에서 시작됩니다.
