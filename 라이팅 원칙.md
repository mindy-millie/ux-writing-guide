# 라이팅 원칙

> **담당자**: @Sally (샐리/AX디자인팀)
>
> 밀리의서재에서 UX 라이팅을 작성할 때 반드시 지켜야 하는 5가지 핵심 원칙입니다.

## 1. 이해하기 쉽게 씁니다

- 일상에서 많이 쓰는 친숙하고 쉬운 표현을 사용합니다.
- 한자어나 영어 같은 외래어 사용을 최소화합니다.
- 전문용어나 내부용어 사용을 최소화합니다.

## 2. 사용자 입장에서 씁니다

- 서비스 공급자가 아닌 사용자 입장에서 용어와 표현을 사용합니다.

| ✅ Do | ❌ Don't |
| --- | --- |
| 혜택 받아가세요. | 혜택을 드립니다. |
| 독서 기록을 볼 수 있어요. | 독서 기록이 제공됩니다. |

- 기능이나 처리 과정보다 사용자가 얻게 되는 결과로 말합니다.

| ✅ Do | ❌ Don't |
| --- | --- |
| 구독이 시작됐어요. | 구독 신청이 완료됐어요. |

## 3. 간결하게 씁니다

- 의미가 없는 표현이나 반복되는 단어, 중복되는 내용은 덜어냅니다.

| ✅ Do | ❌ Don't |
| --- | --- |
| 밀리에서 직접 기획하고 만든 도서 | 밀리에서 직접 기획하고 만든 밀리만의 도서 |

- 과도한 수식어를 사용하지 않습니다.

| ✅ Do | ❌ Don't |
| --- | --- |
| 밀리 오리지널 콘텐츠 | 밀리에만 있는 밀리 오리지널 콘텐츠 |

- 한 문장에는 하나의 내용만 담고, 내용이 길어지는 경우 문장을 나누어 가독성을 높입니다.

## 4. 긍정적으로 씁니다

- 부정적인 상황에서도 할 수 없는 일보다 지금 할 수 있는 행동을 먼저 안내합니다.
- 강요하거나 압박하는 표현은 사용하지 않습니다.

| ✅ Do | ❌ Don't |
| --- | --- |
| 지금 시작하면 혜택을 받을 수 있어요. | 지금 시작하지 않으면 혜택을 놓쳐요. |
| 설정을 건너뛸까요? | 정말 건너뛰시겠어요? |

- 사용자를 탓하거나 죄책감을 느끼게 만드는 표현은 사용하지 않습니다.
- 성별, 연령, 지역, 장애, 사회적 지위를 나타내거나 차별하는 표현은 사용하지 않습니다.

## 5. 명확하게 씁니다

- 행위의 주체와 결과가 바로 보이도록 능동형 문장을 기본으로 사용합니다.

| ✅ Do | ❌ Don't |
| --- | --- |
| 책을 담았어요. | 책이 담겼어요. |
| 화면이 보입니다. | 화면이 보여집니다. |

- 이중 피동 표현은 문장을 어색하게 만들 수 있으므로 사용하지 않습니다.

| ✅ Do | ❌ Don't |
| --- | --- |
| 읽히지 않는 책 | 읽혀지지 않는 책 |

- 피동형 문장은 앞으로 일어날 상황을 설명하거나 피동형이 더 자연스러운 경우에만 사용합니다.

| ✅ Do |
| --- |
| 결제가 완료되면 구독이 시작돼요. |
| 어두운 배경에서도 편하게 읽혀요. |

- 사용자가 핵심을 빠르게 알 수 있도록 중요한 내용은 문장 앞쪽에 배치합니다.

| ✅ Do | ❌ Don't |
| --- | --- |
| 오프라인에서 읽으려면 먼저 다운로드 해야 해요. | 먼저 다운로드를 해야 오프라인에서 읽을 수 있어요. |
